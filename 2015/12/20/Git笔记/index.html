Git笔记

Git笔记

[toc]

0. 初始化配置

配置SSH key ssh-keygen -t rsa -b 4096 -C wenmingvs@gmail.com

git config –global user.name “wenmingvs”
git config –global user.email “wenmingvs@gmail.com”

1. 提交代码到本地与远端

git init
add .ignore
git add .
git commit -m “first commit”
git remote add origin git@github.com:wenmingvs/AcitionBarListView.git
git push -u origin master

2. .ignore文件无效

git rm -r –cached .
git add .
git commit -m “Refresh adding .gitignore file.”

3. 其他命令

git branch //列出分支
git checkout master//切换分支
git status//查看文件变更

4. 设置Git代理

git config –global http.proxy ‘socks5://127.0.0.1:1080’
git config –global https.proxy ‘socks5://127.0.0.1:1080’

5. 分支管理

查看分支：git branch
创建分支：git branch name
切换分支：git checkout name
创建+切换分支：git checkout -b name
合并某分支到当前分支：git merge name
删除分支：git branch -d name
删除远程分支：git push origin :name
撤销未提交的修改：git checkout src/com/android/…/xxx.java
