使用Privoxy转换SSH到HTTP代理

使用Privoxy转换SSH到HTTP代理

  1. 下载Privoxy

  2. 编辑配置文件

sudo vim /usr/local/etc/privoxy/config
forward-socks5 / 127.0.0.1:7070 .

  1. 搜索“listen-address”找到下面这一行，如果你想定义HTTP代理的端口（默认8118），在此修改：
    listen-address 127.0.0.1:8080

  2. 重启
    sudo /Applications/Privoxy/stopPrivoxy.sh
    sudo /Applications/Privoxy/startPrivoxy.sh

  3. netstat -an | grep 8118
